# lein-fortune

An simple webapp, to show fortune cookies

## Usage
lein.sh clean 

lein.sh deps 

lein.sh compile 

lein.sh ring uberjar 

To start the generated artefact: 

<code>java -jar target/lein-fortune-[version]-standalone.jar</code>

Then go to 

http://localhost:3000/fortune
